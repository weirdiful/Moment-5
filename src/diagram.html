Fördelar

En fördel som jag verkligen gillar och saknar i vanlig CSS är nesting, det gör det mycket enklare att skriva tydlig kod och få en bra struktur på den.

Samt mixins och liknande som gör att man har exempelvis text storlek och annat som man ofta återanvänder, vilket också gör det enklare att ändra för hela sidan eftersom man bara behöver ändra mixin och inte varenda elementet i sin CSS kod.

Överlag så finns det många olika funktioner i sass som verkligen underlättar kodningen.

Nackdelar

En nackdel är att det kräver kompilering, att man måste använda sig av något verktyg såsom node eller liknande för att kunna använda SASS.

Även om en del funktioner är relativt enkla och lika CSS så finns det andra funktioner som kan vara svåra att lära sig.

Tankar

Det var roligt att börja lära sig använda SASS och jag tyckte det undelrättade många aspekter av kodningen.

Speciellt så gillar jag nesting och mixins som gjorde det enklare att strukturera upp koden och allt som gjorde att man slipper upprepa sig själv för mycket.

Hoppas de lägger till så man även kan göra samma sak med pseudo-classes i framtiden.
